🚀 使用說明

💡 如何測試新功能
  1. 啟動 React 應用程式
  2. 進入事實查核頁面
  3. 輸入查詢並執行搜索
  4. 在分析結果中點擊「查看整體分析」按鈕
  5. 查看各個模型的詳細分析結果

✨ 新功能特色

  • 整體結果摘要 - 一目了然的查證結果和可信度
  • 權重計算詳情 - 顯示所有模型的評分細節
  • 辯論觀點展示 - 清晰呈現正反雙方的論點
  • 響應式設計 - 支援各種螢幕尺寸
  • 視覺化評分 - 直觀的進度條和標籤顯示
